## 2. Dados Pessoais Recolhidos

Recolhemos os seguintes tipos de dados pessoais, dependendo da sua interação connosco:

## 3. Finalidade do Tratamento de Dados

Os seus dados pessoais são recolhidos e tratados para as seguintes finalidades:

*   **Processamento e Gestão de Encomendas:** Para gerir o seu pedido, processar o pagamento, produzir e enviar os produtos personalizados.
*   **Personalização de Produtos:** Para criar os produtos de acordo com as suas especificações e aprovação.
*   **Comunicação com o Cliente:** Para responder às suas questões, fornecer orçamentos, informar sobre o estado da sua encomenda e prestar apoio ao cliente.
*   **Marketing Direto:** Para enviar newsletters e informações sobre os nossos produtos e promoções, apenas com o seu consentimento prévio e explícito.
*   **Cumprimento de Obrigações Legais:** Para cumprir com as obrigações legais e regulamentares aplicáveis (ex: faturação, obrigações fiscais).

## 4. Fundamento Jurídico para o Tratamento

O tratamento dos seus dados pessoais baseia-se nos seguintes fundamentos jurídicos:

*   **Execução de um Contrato:** Quando o tratamento é necessário para a execução de um contrato de compra e venda ou para diligências pré-contratuais a seu pedido (ex: processamento de encomendas, orçamentos).
*   **Consentimento:** Quando nos dá o seu consentimento explícito para uma finalidade específica (ex: subscrição de newsletter).
*   **Interesse Legítimo:** Quando o tratamento é necessário para os nossos interesses legítimos, desde que não prevaleçam sobre os seus direitos e liberdades fundamentais (ex: melhoria dos nossos serviços, prevenção de fraude).
*   **Obrigação Legal:** Quando o tratamento é necessário para o cumprimento de uma obrigação legal a que estamos sujeitos.

*   **Transportadoras:** Partilhamos os dados estritamente necessários (nome, morada, contacto) com as empresas de transporte para efeitos de entrega das suas encomendas.

## 6. Conservação dos Dados

Os seus dados pessoais são conservados apenas durante o período necessário para cumprir as finalidades para as quais foram recolhidos, ou conforme exigido por lei. Após este período, os dados serão eliminados ou anonimizados de forma segura.
